# 高德地图关键字搜索 POI API 参考

> 官方文档: https://amap.apifox.cn/api-14633570

## 接口信息

- **接口路径**: `GET /v3/place/text`
- **基础 URL**: `https://restapi.amap.com/v3/place/text`

## 请求参数

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|------|------|------|------|
| key | ✅ 是 | string | 高德地图 Web 服务 API Key | - |
| keywords | ✅ 是 | string | 搜索关键词 | `北京大学` |
| types | 否 | string | POI 类型编码，多个用 `\|` 分隔 | `090100` |
| city | 否 | string | 查询城市（名称/citycode/adcode） | `北京` |
| citylimit | 否 | boolean | 是否限制城市范围搜索 | `false` |
| children | 否 | int | 是否返回子 POI（1=返回） | `1` |
| offset | 否 | int | 每页返回数量（1-25，默认20） | `20` |
| page | 否 | int | 当前页码（默认1） | `1` |
| extensions | 否 | string | 返回信息控制 `base`/`all` | `all` |
| output | 否 | string | 响应格式 `JSON`/`XML` | `JSON` |

## 返回字段

### 顶层字段

| 字段 | 说明 |
|------|------|
| status | `1` 表示成功，`0` 表示失败 |
| info | 状态描述 |
| infocode | 状态码，`10000` 表示正常 |
| count | 搜索结果总数 |
| suggestion | 建议关键词和城市 |
| pois | POI 列表（数组） |

### POI 对象字段

| 字段 | 说明 |
|------|------|
| id | POI 唯一标识 |
| name | 名称 |
| type | 类型（如：科教文化服务;学校;高等院校） |
| typecode | 类型编码 |
| address | 地址 |
| location | 经纬度坐标（`经度,纬度`） |
| tel | 电话 |
| pname | 省份名称 |
| cityname | 城市名称 |
| adname | 区/县名称 |
| pcode | 省份行政编码 |
| citycode | 城市行政编码 |
| adcode | 区县行政编码 |
| business_area | 所属商圈 |
| children | 子节点数据（附属设施） |

### 常见错误码

| infocode | 说明 |
|----------|------|
| 10000 | 请求正常 |
| 10001 | key 不正确或过期 |
| 10003 | 访问已超出日访问量 |
| 10004 | 单位时间内访问过于频繁 |
| 20000 | 请求参数非法 |
| 20800 | 规则不正确 |

## API Key 获取

1. 访问 [高德开放平台控制台](https://console.amap.com/)
2. 注册/登录高德开发者账号
3. 创建应用 → 添加 Key → 选择 **Web服务** 类型
4. 获得 API Key 即可使用
